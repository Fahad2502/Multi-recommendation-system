# 🎬🎵📚 Cross-Domain Recommendation System

A Flask-based web application that recommends **Movies, Music, and Books** based on a given movie name.  
This project integrates data from **TMDB API, Spotify API, and Google Books API** to deliver cross-domain recommendations.

---

## 🚀 Features
- 🔍 **Movie Recommendations** using TMDB (The Movie Database)
- 🎵 **Music Suggestions** based on the movie name (Spotify API)
- 📚 **Book Recommendations** related to the movie (Google Books API – public)
- 🎭 **Movie Details** including cast, ratings, and trailer
- 🌐 Built with **Flask**, **Spotipy**, and **RapidFuzz**

---

## 🛠️ Tech Stack
- **Backend**: Flask (Python)
- **APIs**: TMDB, Spotify, Google Books
- **Libraries**: 
  - `spotipy` (Spotify API client)
  - `requests`
  - `python-dotenv` (for managing API keys securely)
  - `rapidfuzz` (for fuzzy matching of titles)
